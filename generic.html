POWER BI
Employee Attrition

From a kaggle data set of IBM HR Analytics of Employee Attrition.

This is a fictional data set created by IBM data scientists. You can check the link for the data set here: https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset

The main goal of this report is to be able to show the employee attrition rate by department and their corresponding employee profiles such as age, marital status, no. of working years, job roles, and monthly rate class. By visualizing this items, we can show an overview of the company's attrition and relate items wherein attrition rates are high to be able to make data driven decisions with regards to employee retention programs.

Data cleaning and manipulation for this report was done through power query. Data visualizations, ofcourse, is through Power BI.
